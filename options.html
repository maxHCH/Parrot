Parrot 設定

查字（Claude）

填入 Anthropic API key，就能拿到中文意思、KK 音標、句中語感和例句。留空的話只會有基本翻譯和 IPA 音標。

學習記錄（Language Quest）

用 Parrot Nest 的帳號登入。每完成一輪循環自動記 XP，累積進等級、連續天數和成就；生詞本（存字、字幕高亮）也走這個帳號，沒登入存不了字。

已登入：

Parrot Nest（web 端）

部署好 Parrot Nest 後把網址填進來，查字小卡會多「去複習」、popup 的今日 XP 會直達 /stats。

每日開口（ChatGPT）

每天第一次 focus Chrome 會跳通知推你去 ChatGPT。只計 ChatGPT 分頁在前景的時間，達標 streak +1，並記進 Language Quest。

⚠️ API key 和登入 token 都存在 chrome.storage.local，只在你自己的瀏覽器裡。不要把這個資料夾連 key 一起分享或上傳。 查一個字大約用掉 300～500 tokens，用 Haiku 幾乎不花錢。
